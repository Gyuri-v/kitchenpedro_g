포인트 적립/사용/소멸
1. 결제금액의 3% 적립 (부가세 제외, 할인내역 제외, 무료 식사권, 쿠폰, 금액 할인권, 도시락 구매, 당사에서 지정한 제외 상품 등 적립 불가)
2. 적립된 포인트는 3,000p부터 10p 단위로 사용 가능 (기타 할인 혜택과 중복 사용 가능하며, 결제 시 포인트 선 차감 후 타 할인이 적용)
3. 적립된 포인트 사용 시, 포인트 사용 비밀번호 숫자 6자리 입력 후 사용 가능
4. 포인트 사용 비밀번호 분실 시, 마이아웃백 > 개인정보 수정에서 포인트 사용 새 비밀번호 설정 가능
5. 매장에서 실물 부메랑 카드 또는 모바일 부메랑 카드를 제시 후 포인트 사용 가능 (스마트폰 사용자의 경우 아웃백 어플을 설치하여 부메랑 카드 번호 조회 가능)
6. 현장에서 결제 시 부메랑 포인트를 적립 받지 못한 경우 결제일로부터 14일 이내에 홈페이지 마이 아웃백 > 포인트 사후적립 메뉴에서 적립
7. 포인트는 포인트 적립 연도 기준 다음해 12월 31일 24시에 일괄 소멸
